# Secured-Pi
Code 401 (Python) - final project

## Description
Python web application for a smart lock powered by Raspberry Pi; features facial recognition for security.

## Setup instructions:
```
python3 -m venv ENV
source ENV/bin/activate
pip install -r requirements.txt
```
